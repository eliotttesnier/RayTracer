### Commit norm

Template for the commits:

    [type] content: description

Types of commits:

    - [ADD]     ->  addition
    - [UPT]     ->  update
    - [CI]      ->  Modify CI
    - [FIX]     ->  correction
    - [REFACTO] ->  refacto
    - [MERGE]   ->  PR merge
    - [RM]      ->  removed content
